Certifications

These are the certifications I have earned throughout my time in the Information Technology field.

IT Specialist - Artificial Intelligence

Candidates for this exam are AI enabled learners and are prepared for the professional use of AI by understanding how AI can be used to solve problems.
IT Specialist - Cloud Computing

Candidates for this exam will be able to identify and describe end solutions that leverage cloud technologies, considerations that span across solutions and the "art of the possible" in utilizing cloud to develop solutions.
IT Specialist - Computational Thinking

Candidates for this exam decompose problems, collect and analyze data, recognize patterns in data, represent data through abstractions, and automate solutions by using algorithmic thinking.
IT Specialist - Cybersecurity

This exam assesses understanding of key security paradigms, terminology, and mindset. Badge earners have a keen awareness of the importance of security and the threats to a business when security procedures are not followed.
IT Specialist - Data Analytics

Candidates for this exam are seeking to prove introductory knowledge of how to responsibly manipulate, analyze, and communicate findings of data analysis. They should be familiar with general data concepts, data-related laws, and responsible analytics practices.
IT Specialist - Databases

Earners of this badge demonstrate foundational knowledge of how to design and query relational databases, such as MySQL, Microsoft SQL Server, or Oracle.
IT Specialist - Device Configuration and Management

Earners of this badge demonstrate foundational skills in, and a basic knowledge of, Windows devices and mobility.
IT Specialist - HTML and CSS

Earners for this badge demonstrate that they can structure a webpage using HTML elements and create and apply styles using CSS.
IT Specialist - HTML5 Application Development

Earners of this badge demonstrate their ability to use HTML5, CSS, and JavaScript to build responsive web applications that will run on a variety of touch-enabled devices, including PCs, tablets, and phones.
IT Fundamentals Pro

The TestOut IT Fundamentals Pro certification validates foundational IT skills, focusing on computer hardware, software, networking, databases, programming, and cybersecurity. It's designed to provide students with a solid understanding of essential IT concepts.
IT Specialist - Java

Earners of this badge demonstrate that they can recognize, write, and debug Java code that will logically solve a problem.
IT Specialist - JavaScript

Earners of this badge demonstrate that they can recognize, write, and debug JavaScript code that will logically solve a problem.
IT Specialist - Networking

Earners of this badge demonstrate foundational networking knowledge and skills, including TCP/IP, networking services, networking topologies, and troubleshooting in wired and wireless environments.
IT Specialist - Network Security

Earners of this badge demonstrate foundational security knowledge and skills, including an understanding of core security principles operating system security, network and device security, and secure computing practices.
OSHA 10-Hour

OSHA 10-hour training is a safety course designed by the Occupational Safety and Health Administration (OSHA) to provide entry-level workers with basic safety and health information.
IT Specialist - Python

Earners of this badge demonstrate that they can recognize, write, and debug Python code that will logically solve a problem.
